This package is forked from [django-jsonform](http://django-jsonform.rtfd.io/)



## Install

Install via pip:

```sh
$ pip install django-adminform
```

Edit your *settings.py* file:

```python
# settings.py

INSTALLED_APPS = [
    # ...
    'django_adminform'
]
```
